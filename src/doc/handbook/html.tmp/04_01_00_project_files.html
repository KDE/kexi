4.1. Project files

Many applications such as OpenOffice.org or Microsoft Excel create files which are called documents. Kexi also creates files but we will call them Kexi projects or Kexi database files.

Notes

  • Kexi database can also be stored on database servers. THis is why, to avoid ambiguity, we talked above about Kexi database files and not Kexi database. The latter will be used however if the way you store the database is irrelevant.
